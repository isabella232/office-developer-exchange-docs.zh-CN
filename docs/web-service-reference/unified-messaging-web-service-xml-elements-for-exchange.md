# <a name="unified-messaging-web-service-xml-elements-for-exchange"></a>统一消息 Web 服务 XML 元素Exchange

查找属于统一消息和 UM web 服务一 (UM) WEB 服务的 XML Exchange。
  
统一消息 (UM) Web 服务包含许多 XML 元素，这些元素用于定义在客户端和服务器之间发送的请求和响应。 本节提供有关各个 XML 元素的信息。
  
> [!NOTE]
> 对于 Exchange 从 Exchange 2010 开始的版本，建议使用[Exchange Web 服务 (EWS) ](https://msdn.microsoft.com/library/60285497-0c4e-4e51-84e1-34dd6d89a5d8%28Office.15%29.aspx)中提供的 UM 操作，而不是 UM Web 服务，原因如下： 
> - 基于 EWS 的 UM 功能在 EWS 托管 API 中具有一流支持。 
> - 在 2010 Exchange 2010 Exchange版本中，所有新的 UM 功能都添加到 EWS，而不是 UM Web 服务。
