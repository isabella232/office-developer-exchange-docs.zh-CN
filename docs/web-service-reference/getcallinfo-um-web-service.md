# <a name="getcallinfo-um-web-service"></a>GetCallInfo（UM Web 服务）

**GetCallInfo** 元素定义一个请求，用于获取有关调用的信息。 
  
[GetCallInfo（UM Web 服务）](getcallinfo-um-web-service.md)
  
```xml
<GetCallInfo>
  <CallId/>
</GetCallInfo>
```

 **complexType**

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。

|**元素**|**说明**|

无。
