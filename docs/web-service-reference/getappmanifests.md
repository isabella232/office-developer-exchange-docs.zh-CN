# <a name="getappmanifests"></a>GetAppManifests

**GetAppManifests** 元素是请求返回应用清单的基元素。 
  
```xml
<GetAppManifests />
```

 **GetAppManifestsType**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

[ApiVersionSupported](apiversionsupported.md)  | [SchemaVersionSupported](schemaversionsupported.md)
  
### <a name="parent-elements"></a>父元素

无。
  
## <a name="text-value"></a>文本值

无。
  
## <a name="remarks"></a>注解

**GetAppManifests** 元素适用于从 Exchange Online 2013 Microsoft Exchange Server面向 Exchange 版本的客户端。
