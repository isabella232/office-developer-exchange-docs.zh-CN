# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

**AlternateMailboxes** 元素表示备用邮箱的集合。 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
