# <a name="meetingsuggestion"></a>MeetingSuggestion

**MeetingSuggestion**元素指定建议的会议。 
  
```XML
<MeetingSuggestion>
   <Attendees/>
   <Location/>
   <Subject/>
   <MeetingString/>
   <StartTime/>
   <EndTime/>
</MeetingSuggestion>
```

 **MeetingSuggestionType**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

[与会者](attendees.md)  | [位置](location.md)  | [主题](subject.md)  | [Meetingsuggestion.meetingstring](meetingstring.md)  | [StartTime](starttime.md)  | [EndTime](endtime.md)
  
### <a name="parent-elements"></a>父元素

[MeetingSuggestions](meetingsuggestions.md)
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
