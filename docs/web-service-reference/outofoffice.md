# <a name="outofoffice"></a>OutOfOffice

**OutOfOffice** 元素表示响应消息和发送响应消息的持续时间。 
  
```XML
<OutOfOffice>
   <ReplyBody/>
   <Duration/>
</OutOfOffice>
```

```XML
<OutOfOffice>
   <ReplyBody/>
</OutOfOffice>
```

**OutOfOfficeMailTip**

## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

|**元素**|**说明**|
|:-----|:-----|
|[ReplyBody](replybody.md) <br/> |包含 Out of Office (OOF) 邮件和用于邮件的语言。  <br/> |
|[Duration (UserOofSettings)](duration-useroofsettings.md) <br/> |包含 OOF 状态在 [OofState](oofstate.md) 元素设置为 Scheduled 时启用的持续时间。  <br/> |
   
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[邮件提示](mailtips.md) <br/> |表示各种类型的邮件提示的值。  <br/> |
   
## <a name="text-value"></a>文本值

无。
  
## <a name="remarks"></a>注解

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
