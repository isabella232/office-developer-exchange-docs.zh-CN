# <a name="openasadminorsystemservice"></a>OpenAsAdminOrSystemService

**OpenAsAdminOrSystemService** 元素仅供内部使用。 此元素不由客户端使用。 
  
```XML
<OpenAsAdminOrSystemService LogonType="">
   <ConnectingSID/>
</OpenAsAdminOrSystemService>
```

 **OpenAsAdminOrSystemServiceType**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

|**属性**|**说明**|
|:-----|:-----|
|LogonType  <br/> |不适合客户端使用。  <br/> |
   
### <a name="child-elements"></a>子元素

|**元素**|**说明**|
|:-----|:-----|
|ConnectingSID  <br/> |不适合客户端使用。  <br/> |
   
### <a name="parent-elements"></a>父元素

无。
  
## <a name="text-value"></a>文本值

无。
  
## <a name="remarks"></a>注解

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。Exchange Server 2010 Service Pack 1 (SP1) 中引入了此元素。
