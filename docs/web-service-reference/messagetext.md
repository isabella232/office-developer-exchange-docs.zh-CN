# <a name="messagetext"></a>MessageText

**MessageText** 元素提供响应状态的文本说明。 
  
```XML
<MessageText/>
```

 **string**

## <a name="remarks"></a>注解

此元素不是必需的，并不包含在所有响应中。 返回错误消息时包含此元素。 
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
