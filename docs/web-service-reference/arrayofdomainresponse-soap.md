# <a name="arrayofdomainresponse-soap"></a>ArrayOfDomainResponse (SOAP)

**ArrayOfDomainResponse** 元素包含每个请求域设置的响应数组。 
  
```XML
<ArrayOfDomainResponse>
   <DomainResponse/>
</ArrayOfDomainResponse>
```

 **ArrayOfDomainResponse**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。

|:-----|:-----|
