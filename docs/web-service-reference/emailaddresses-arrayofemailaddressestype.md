# <a name="emailaddresses-arrayofemailaddressestype"></a>EmailAddresses (ArrayOfEmailAddressesType)

**EmailAddresses** 元素指定关联人物的所有电子邮件地址的数组。 
  
```XML
<EmailAddresses>
    <Address></Address>
</EmailAddresses>
```

 **ArrayOfEmailAddressesType**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

|**元素**|**说明**|
|:-----|:-----|
|[地址 (EmailAddressType)](address-emailaddresstype.md) <br/> |表示一个完全解析电子邮件地址。  <br/> |
   
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[角色](persona.md) <br/> |指定由 **GetPersona** 请求返回的一组人员数据。  <br/> |
   
## <a name="remarks"></a>注解

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
